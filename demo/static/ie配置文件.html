ie配置方法

第一步： 配置 babel-polyfill
	1: 在package.json 中引入
		"devDependencies":{
			"babel-polyfill": '^6.26.0',
		}
	2: 在main.js中引入
		import 'babel-polyfill'
	3：在vuex中的index中引入
		import 'babel-polyfill'
	4: 在build文件夹中，webpack.base.conf.js引入
		entry: {
		 app: ["babel-polyfill", "./src/main.js"]
		}

第二步: 将要替代的iview引入到node_modules,
	注意： package.json中iview的版本要跟引入的一致

	截至到这个时候可以试试运行看看，如果有问题 再执行下面的部分

第三步： 解决promise未定义的问题

	1：安装babel-preset-es2015
		将包放入node_modules， 并在在package.json 中引入
		"devDependencies":{
			"babel-preset-es2015": '^6.24.1',
		}
	2. 在.babelrc文件中引入
		{
			"presets" [
				"stage-2",
				"es2015"
			],
		}
